# ADONIS_ON_DEMAND_REST_API

## Setup

Clone the repo then install dependencies :

```bash
npm install
```

Create .env file :

```bash
cp .env.example .env
```

### Migrations

Run the following command to run startup migrations.

```js
adonis migration:run
```

### Start

Run the following command to start the HTTP server (dev mode).

```bash
npm run dev
```
